Welcome to the Universal Media Database

Keep track of all your media. Add movies, TV shows, games, books, music and art to your master list. Rate each item you add and keep track of your start and finish dates. Discuss movies, episodes, games, songs, and etc. with other users.


Add media to custom lists consisting of one or more different types (e.g. a "Best Psychological Horror" could contain your top picks from movies, games, books, and TV shows) and share these lists with other users or on social media.


Read and write reviews, view and sort media by average user rating, and get recommendations based on your highly rated items.

Currently implemented features:

  • Users can create a new account and log in.
  • Users can seach for movies and people.
  • Users can add movies to their masterlist.
  • Users can rate and review movies.
  • Users can add personal information about a movie in their masterlist (such as when they watched it and how many times they have rewatched it).
  • Users can create new public movie entries that they and other users can add to their lists.
  • Users can create new contributors (e.g. actors and directors) entries and attribute them to one or more movies.
  • Users can edit movie and contributor entries.
  • User ratings are averaged and displayed on the entry page.
  • Top rated movies are displayed on the home page.
  • Administrators can soft or hard delete entries.
  • Administrators can search for and edit or restore soft deleted entries.

High priority planned features:

  • A profile page where users can see their own profile all of their own lists and information, as well as other user's profiles.
  • Pagination and query restrictions for searches—for scalability and to prevent unnecessary server traffic.
  • Pagination on automatically generated lists.
  • Ability to add character roles to actors.
  • Ability for users to create custom lists with unique, sharable URLs.
  • Ability to share lists and reviews on social media.
  • Ability to sort by column and search within any list.
  • Ability to browse media by type.
  • Many general UI and ease of use enhancements, such as a warning when attempting to navigate away from an imcomplete form.
  • General visual enhancements, including properly responsive pages.
  • Search on enter when adding or editing entries.
  • More efficient search algorithm—get rid of .ToLower's, and replace nested loops with hashtable lookups.
  • More efficient handling of triple nested objects—prevent duplicates when adding nested objects without removing objects and re-adding them by reference.
  • Ability for adiminstrators to search for and restrict, ban, or permanently delete users.
  • Email verification for new user accounts.
  • Recording of user IDs when a user edits an entry—to prevent abuse.
  • Content checking and sanitization of user input to combat malicious users.

Future features:

  • Implementation of other media types (TV shows, games, music, books, and art).
  • Automatic retrieval of metadata when adding a new entry (using APIs such as Rotten Tomatoes and Musicbrainz).
  • Ability to comment on entries and reviews.
  • Ability to automatically add entire entries to the database using APIs.
  • Private messaging system.
  • Episode/chapter/movie discussions.
  • Full discussion board.
  • Ability to write recommendations/comparitive reviews.
  • Automatic recommendations based on the ratings of items in users' lists.
